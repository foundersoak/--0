# Attribution & data sources

**Undefeated** is an unofficial fan game. It is not affiliated with, endorsed by, or sponsored by
the NBA, NFL, MLB, NHL, NCAA, the Premier League, or any team. Team names and colors are used
nominatively to identify franchises; no team logos or player likenesses are used.

## Player data

The shipped player data is **hand-curated for the game** (peak-era statistical averages authored from
public reference material). Individual statistical facts are not copyrightable.

As the project grows, datasets will be built (at build time, then committed) from permissively
licensed open sources, with attribution preserved here:

- **MLB** — the [Lahman Baseball Database](https://sabr.org/lahman-database/) (CC BY-SA 3.0).
- **NFL** — [nflverse](https://github.com/nflverse) open data.
- **NHL** — [MoneyPuck](https://moneypuck.com/data.htm) and open Kaggle datasets.
- **Soccer** — [StatsBomb Open Data](https://github.com/statsbomb/open-data).
- **NBA / CFB** — open Kaggle datasets plus curated backfill.

We do **not** scrape or redistribute Sports-Reference data; those sites are used only as a human
reference when authoring curated entries.

CC BY-SA-derived datasets, where used, are kept in clearly marked files and shared under the same
license; this obligation does not extend to the application code.
